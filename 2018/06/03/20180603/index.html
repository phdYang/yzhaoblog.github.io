发表于 2018-06-03 | 更新于: 2018-06-03 | 分类于 语言基础 | | 本文阅读量
字数统计: 1,628 | 阅读时长 ≈ 8

循环结构，是编程语言中最重要的结构，因为循环，我们才能写出复杂的程序，也是因为循环，我们的程序才能够实现所谓的代替重复性质的手工劳作而得以实现自动化。

所以，循环的本质就是代替重复性质的工作。

本文找寻了十几道典型的循环例题，每题去考虑如何使用for和while两种方式去实现。

用以帮助程序初学者理解，和自己回忆。

结尾会放置一道面试常问的顺时针打印数组的问题（也就是传说中的蛇形填数）。

思考：本题目的while循环比for循环要简单。因为，这不是典型的遍历类循环，而是条件类循环。所以可以看到for循环的结束其实是受到循环内if的控制。而while循环本身就是循环和条件的统一体。

合理选用for和while也是需要练习的，但从表达能力上来看，两者是可以等价的。

总结：单层循环没有什么需要练习的，非常基础。所以可以看到，单层循环需要和if等判断语句结合来达到使程序复杂的目的。
